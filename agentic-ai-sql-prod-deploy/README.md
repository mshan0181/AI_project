# Agentic AI - Production-ready Deployment Package

This repository contains a production-ready deployment scaffold for an Agentic AI system that:
- Translates natural language to SQL using an LLM
- Executes SQL against MySQL (MCP FastAPI)
- Provides a Gradio UI for users
- Integrates with LangSmith for tracing
- Includes Docker Compose + nginx-proxy + Let's Encrypt and Kubernetes manifests for auto-scaling

See folders for details.
